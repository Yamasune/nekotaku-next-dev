如果遇到启动程序时提示缺少 DLL 或无法执行代码等错误等通常是由于您当前的 Windows 系统缺少了对应的 DLL 模块所导致，我们推荐您使用 DirectX 修复工具（DirectX Repair），这是一款一键修复 DirectX、C++、API Sets 的工具软件

## 下载方法

您可以从作者的 CSDN 博客中获得下载链接：[DirectX修复工具增强版](https://blog.csdn.net/vbcom/article/details/7245186)

如果下载遇阻也可以在本站网盘下载：[DirectX_Repair(Enhanced_Edition).7z](https://www.nullcloud.top/Software/@VBcom/DirectX_Repair/DirectX_Repair(Enhanced_Edition).7z)

## 使用方法

先解压下载好的修复工具压缩包，并在解压出来的文件夹内双击 DirectX Repair(.exe) 来启动程序

![](https://image.nullcloud.top/uploads/posts/repair/20241210091800.avif)

在程序中单击 “检测并修复” 旁边的箭头勾选 “同时更新c++” 后再点击 “检测并修复”

![](https://image.nullcloud.top/uploads/posts/repair/20241210092124.avif)

此时程序会检测您 Windows 中缺失的 DLL 模块并进行修补，完成后请手动重启一遍计算机以确保修复生效

## 注意事项

如果您的 Windows 先前已修复过 C++ 运行库，但是我们仍建议您进行二次修复，因为 C++ 运行库一直在更新，先前更新的运行库可能存在不完整现象，目前修复程序已支持到了 Visual C++ Runtime 2022

程序兼容 32 位以及 64 位作业系统，如果您的计算机处于 Windows XP 系统，请先完成 Microsoft .NET Framework 2.0 运行库的安装，否则可能无法运行此程序

程序运行时可能会在右侧底部出现广告弹窗，此为程序作者添加的广告服务，与本站无关，此广告并不会导致您的计算机出现异常，如对此反感请点击弹窗右上角的关闭 (X) 即可
